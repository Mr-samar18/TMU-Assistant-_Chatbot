
TMU College Chatbot is a web-based academic chatbot developed to help students quickly access important university-related information such as courses, admissions, fees, hostel facilities, placements, and general campus details.

This project is built as a college academic project using Python, Flask, HTML, CSS, and JavaScript.




The chatbot is designed to answer frequently asked questions related to Teerthanker Mahaveer University (TMU). It uses a question-answer matching system with fuzzy text matching to understand user queries and provide the most relevant response.

The goal of this project is to create a simple, user-friendly, and interactive student support assistant that can help users explore university information in an easy conversational format.



- Interactive chatbot interface
- TMU-related question answering
- Information about courses and programs
- Admission-related guidance
- Fees and eligibility support
- Hostel and campus facility information
- Placement-related information
- Dark mode support
- Responsive modern UI
- Typing indicator and chat timestamps
- Fuzzy matching for better question understanding



- Python
- Flask
- RapidFuzz

- HTML
- CSS
- JavaScript
